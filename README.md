# BW Premium Portfolio

React + Vite + Tailwind CSS + Framer Motion + GSAP + Lenis + React Three Fiber.

## Ishga tushirish
```bash
npm install
npm run dev
```

## Telegram formani lokal sinash
```bash
npm install -g vercel
```
`.env.local` yarating:
```env
TELEGRAM_BOT_TOKEN=YANGI_MAXFIY_TOKEN
TELEGRAM_CHAT_ID=6570315540
```
Keyin:
```bash
vercel dev
```

## Vercel
GitHub repository'ni Vercel'ga import qiling va Settings → Environment Variables ichida `TELEGRAM_BOT_TOKEN` hamda `TELEGRAM_CHAT_ID` ni qo‘shing. Tokenni GitHub'ga yuklamang.
